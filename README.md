# Book Management API

API untuk manajemen buku dan kategori menggunakan Golang dan PostgreSQL.

## Endpoints
- `GET /api/categories`
- `POST /api/categories`
- `GET /api/categories/:id`
- `DELETE /api/categories/:id`
- `GET /api/categories/:id/books`
- `GET /api/books`
- `POST /api/books`
- `GET /api/books/:id`
- `DELETE /api/books/:id`
